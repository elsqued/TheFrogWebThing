# 🐸 Project Notice & Documentation

This official Frog website is built using a modern-retro **Aero Glass** aesthetic, inspired by the skeuomorphic designs of the mid-2000s.

## 🛠 Technology Stack
This project is designed to be lightweight and fast, running entirely in the browser:
* **HTML5/CSS3**: Custom glassmorphism and glossy gradients.
* **JavaScript (Vanilla)**: For dynamic content fetching and clipboard management.
* **Marked.js**: Used for real-time Markdown-to-HTML rendering.
    * *Source:* `https://cdn.jsdelivr.net/npm/marked/marked.min.js`

## 📋 Features
1.  **Dynamic Content**: The website automatically pulls the latest updates from our `README.md` via the GitHub API.
2.  **Smart Links**: 
    * Standard external links open automatically in a **new tab**.
    * Our server address (`frogsquadprecombat.aternos.me`) is **automatically copied** to your clipboard upon clicking, instead of opening a page.
3.  **Aero Interface**: High-fidelity reproduction of classic desktop window effects, including blurred backgrounds and glossy title bars.
